# DNNrocket Documentation

Welcome to the DNNrocket documentation.

## Overview

DNNrocket is a powerful Content Management System (CMS) framework for DNN Platform, providing a flexible and extensible foundation for building dynamic web applications.

## Components

### Core Framework
- **DNNrocket Core** - The main API and framework
- **RocketTools** - Development and utility tools
- **RocketUtils** - Common utilities library
- **RocketPortal** - Portal management functionality
- **RocketEndInstall** - Installation and setup utilities
- **AppThemes** - Theme management and templates
- **Simplisity** - Simplification library

### Modules
- **RocketBlog** - Blog management module
- **RocketContent** - Content management module
- **RocketDirectory** - Directory listing module
- **RocketEvents** - Event management module
- **RocketForms** - Form builder module
- **RocketNews** - News management module
