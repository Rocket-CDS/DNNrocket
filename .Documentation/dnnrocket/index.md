# DNNrocket Core Components

DNNrocket provides a comprehensive framework for building powerful DNN applications.

## Core Framework

### [DNNrocket Core](DNNrocket.md)
The main API and framework that powers all DNNrocket modules.

### [RocketTools](RocketTools.md)
Development and utility tools for building and managing DNNrocket applications.

### [RocketUtils](RocketUtils.md)
Common utilities library providing helper functions and shared functionality.

### [RocketPortal](RocketPortal.md)
Portal management functionality for multi-portal environments.

### [RocketEndInstall](RocketEndInstall.md)
Installation and setup utilities for DNNrocket components.

### [AppThemes](AppThemes.md)
Theme management and templates for customizing the look and feel.

### [Simplisity](Simplisity.md)
Simplification library for streamlined development.
